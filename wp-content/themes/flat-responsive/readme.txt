-------------------------------------------------------
  Flat Responsive Features
-------------------------------------------------------

- Responsive
- Unlimited colours
- WordPress customizer for making changes to theme settings
- 16 Widget positions
- Several page templates
- WordPress gallery styling
- Jetpack ready
- Font based icons
- Retina ready graphics for high resolution screens
- Several Post formats
- Various typography elements
- Translation Ready with the .pot file
- WooCommerce Ready
- Widgetkit slideshow supported



-------------------------------------------------------
    Recommended Plugins - Optional
-------------------------------------------------------

To get your site looking like the live demo website, this theme uses several
plugins to help make the enhancements to give your website an 
amazing showcase style.

- Jetpack
- Breadcrumbs NavXT
- Hide Title
- Remove Widget Titles
- Simply Exclude
- Widget CSS Classes
- WP Google Maps
- Font Awesome Shortcodes
- Widgetkit

Please note that if you do not use or plan to use Jetpack, I've noted
a couple plugins to use, otherwise Jetpack already has those built-in.
